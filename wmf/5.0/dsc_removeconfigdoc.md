# <a name="remove-dsc-documents"></a>Supprimer des documents DSC

Quand un document de configuration est remis à DSC, il passe par différentes phases (en attente, en cours, précédent). Nous avons ajouté une nouvelle applet de commande à DSC dans Windows PowerShell 4.0, `Remove-DscConfigurationDocument`, dans le cadre du [correctif cumulatif de novembre 2014 pour Windows RT 8.1, Windows 8.1 et Windows Server 2012 R2](https://support.microsoft.com/kb/3000850).
